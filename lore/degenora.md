# Degenora Island

## Overview

**Location:** Quarantined sector, coordinates classified  
**Status:** Active quarantine - unauthorized entry prohibited  
**Size:** 47 square kilometers  
**Discovery:** 2024, origin unknown

### Notable Features

- **The Signal Tower:** Broadcast source of all transmissions
- **Specimen Containment:** Multiple sites across zones

- **The Breach:** Sealed anomaly site - extreme hazard
