## The Signal

Terminal 7-B broadcasts The Signal on encrypted channels:

- **Frequency:** Variable, locked to FRUG NFT holders
- **Encryption:** Wallet-signed authentication
- **Content:** Live comms, lore drops, specimen alerts

### Transmission Types

1. **Scheduled:** Daily status reports, weekly dossiers
2. **Event:** Specimen discoveries, anomaly alerts
3. **Emergency:** Quarantine breaches, hazard warnings

## Clearance System

Terminal 7-B enforces strict clearance protocols:

### Access Levels

- **C (Civilian):** Public terminals only
- **B (Basic):** Live feeds, basic map
- **A (Advanced):** Archives, specimen data
- **S (Special):** Deep systems, The Breach files

### Upgrade Path

1. Acquire FRUG NFT (Swamp or Frugugu)
2. Verify wallet ownership
3. Sign attestation message
4. Automatic clearance upgrade
